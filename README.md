# MediaProject_Peacock
개인 맞춤형 가계부 서비스 픽콕 프로젝트

## 🔍 프로젝트 소개

* 카드 결제 문자를 소비 카테고리에 맞게 분류합니다.
* 지출 분석 및 예산을 차트로 시각화하여 보여줍니다.
* 소비패턴에 따른 금융서비스를 추천합니다.
